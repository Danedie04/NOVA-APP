INTEGRATION

WIREGUARD IN 4 LINES

Replace the mock tunnel with GoBackend. The VpnService.Builder stays identical.

NovaVpnService.kt — WireGuard integration point
// ── TODO:WIREGUARD ─────────────────────────────────────────
// Replace runTunnel() mock loop with these 4 lines:

val backend = GoBackend(this)
val tunnel = object : Tunnel {
    override fun getName() = "NovaVPN"
    override fun onStateChange(state: Tunnel.State) {
        _vpnState.value = if (state == Tunnel.State.UP)
            VpnState.Connected(serverIp = config.serverIp)
        else VpnState.Disconnected
    }
}
val wgConfig = Config.parse(buildWireGuardConfig(config))
backend.setState(tunnel, Tunnel.State.UP, wgConfig)

// ── VpnService.Builder stays 100% identical ─────────────────
